To install ZBBS properly:

	-unzip ZBBS.ZIP from the root of your C drive
	-setup a system DSN called "zbbsdb" to the 
	 access database located in c:\zbbs\data\zbbs.mdb
	-run zbbsinstall.bat to register zbbsglobal.exe 
	 and zbbsnode.exe
	-ALL DONE!  double-click the shortcut called
	 ZBBS Startup

Happy hacking.
